在未排序的数组中找到第 k 个最大的元素。请注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。

示例 1:

输入: [3,2,1,5,6,4] 和 k = 2
输出: 5
示例 2:

输入: [3,2,3,1,2,4,5,5,6] 和 k = 4
输出: 4
说明:

你可以假设 k 总是有效的，且 1 ≤ k ≤ 数组的长度。

### 大根堆

大根堆（先排序在找到第K大的数）

```java
package leetcode.heap;

public class FindKthLargest {

    public static void main(String[] args) {
        int[] arr = {3, 2, 3, 1, 2, 4, 5, 5, 6};
        System.out.println(findKthLargest(arr, 4));
    }

    public static int findKthLargest(int[] nums, int k) {
        if (nums == null) {
            return 0;
        }
        for (int i = 0; i < nums.length; i++) {
            heapInsert(nums, i);
        }
        //找到最大的第K个数
        if (k == 1) return nums[0];
        int heapSize = nums.length;
        swap(nums, 0, --heapSize);
        while (--k > 0) {
            heapify(nums, 0, heapSize);
            swap(nums, 0, --heapSize);
        }
        return nums[heapSize];
    }

    /**
     * 入堆
     *
     * @param arr
     * @param index
     */
    public static void heapInsert(int[] arr, int index) {
        while (arr[index] > arr[(index - 1) / 2]) {
            swap(arr, index, (index - 1) / 2);
            index = (index - 1) / 2;
        }
    }

    public static void heapify(int[] arr, int index, int heapSize) {
        //(1)找到左孩子
        int left = 2 * index + 1;
        //（2）循环条件是：左孩子坐标小于比较的堆大小
        while (left < heapSize) {
            //(3)如果存在右孩子的情况下，找到左右孩子中的较大值
            int largest = (left + 1) < heapSize && (arr[left + 1] > arr[left]) ? left + 1 : left;
            largest = arr[largest] > arr[index] ? largest : index;
            //（4）判断左右孩子和父节点的最大值
            if (largest == index) {
                break;
            }
            //(5)交换元素
            swap(arr, index, largest);
            //(6)更改循环变量进行下一次循环
            index = largest;
            left = 2 * index + 1;
        }
    }

    public static void swap(int[] arr, int i, int j) {
        int tmp = arr[i];
        arr[i] = arr[j];
        arr[j] = tmp;
    }
}

```

### 小根堆

维护一个小根堆，遍历数组，如果当前元素大于根，则加入堆中；
如果加入后堆的元素数量多于k，则弹出根元素。遍历一遍之后，根就是第k大的元素。

```java
package leetcode.heap;

public class SmallRootHeapFindKthLargest {

    public static void main(String[] args) {
//        int[] arr = {3, 2, 3, 1, 2, 4, 5, 5, 6};
        int[] arr = {3,2,1,5,6,4};
        System.out.println(findKthLargest(arr, 2));
    }

    /**
     * 维护一个小根堆，遍历数组，如果当前元素大于根，则加入堆中；
     * 如果加入后堆的元素数量多于k，则弹出根元素。遍历一遍之后，根就是第k大的元素。
     */

    public static int findKthLargest(int[] nums, int k) {
        if (nums == null) return 0;
        int size = 1;
        //维护一个大小为k的小根堆
        int i = 0;
//        int[] tmpHeap = new int[k];
        for (; i < k; i++) {
            heapInsert(nums,i);
        }
//        System.arraycopy(nums,0,tmpHeap,0,k);
        for (; i < nums.length; i++) {
            if(nums[0] < nums[i]) {
                nums[0] = nums[i];
//                heapInsert(nums,i);
//                swap(nums,0,k);
                heapify(nums,0,k);
            }
        }
        return nums[0];
    }

    /**
     * 入堆
     */
    public static void heapInsert(int[] arr, int index) {
        while (arr[index] < arr[(index - 1) / 2]) {
            swap(arr, index, (index - 1) / 2);
            index = (index - 1) / 2;
        }
    }

    /**
     * 保证小根堆的特性
     */
    public static void heapify(int[] arr, int index, int heapSize) {
        int left = 2 * index + 1;
        while (left < heapSize) {
            int largest = (left + 1) < heapSize && arr[left+1] < arr[left] ? left + 1 : left;
            largest = arr[largest] < arr[index] ? largest : index;
            if (largest == index) {
                break;
            }
            swap(arr, largest, index);
            index = largest;
            left = 2 * index + 1;
        }
    }

    public static void swap(int[] arr, int i, int j) {
        int tmp = arr[i];
        arr[i] = arr[j];
        arr[j] = tmp;
    }
}
```
